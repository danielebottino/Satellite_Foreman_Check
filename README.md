Satellite / Foreman Registration Detection
=========

Playbook For Determining if your VMs are registered to Foreman or Satellite

Requirements
------------

Insert your hosts in the inventory.ini file and change the "hosts:" inside the playbook
!ATTENTION! Remember to Check if your terminal is in ENG language
Ansible-Core Modules (default)

Example Playbook
----------------

    - hosts: testing # Or your host wave
      tasks:
